# AEGIS ERP

> Sistema ERP Full Stack desenvolvido para demonstrar boas práticas de engenharia de software, arquitetura escalável e desenvolvimento web moderno.

## Objetivo

O AEGIS ERP é um sistema completo para gerenciamento empresarial, desenvolvido como projeto de portfólio.

## Funcionalidades Planejadas

- Autenticação de usuários
- Dashboard administrativo
- Gestão de produtos
- Controle de estoque
- Clientes
- Fornecedores
- Compras
- Vendas
- Financeiro
- Relatórios
- Auditoria

## Tecnologias

### Front-end
- HTML5
- CSS3
- JavaScript

### Back-end
- Node.js
- Express

### Banco de Dados
- MySQL
